# Categorie

## Struttura gerarchica

Le categorie sono organizzate su **2 livelli**:
- **Categorie radice** — mostrate in grassetto (es. Salotti e Divani, Sedie, Tavoli)
- **Sottocategorie** — rientrate con il prefisso "└─" (es. └─ Divani angolari)

## Come aggiungere una categoria

1. Vai su **Categorie → Nuova categoria**
2. Inserisci il **nome** (lo slug viene generato automaticamente)
3. Scegli la **categoria padre** se è una sottocategoria (lascia vuoto per una radice)
4. Il campo **Ordine** controlla la posizione nella lista

## Categorie inline

Quando aggiungi o modifichi un prodotto, puoi creare una nuova categoria direttamente
dalla select "Categoria" nel form prodotto — senza uscire dalla scheda.
